# Spring Thymeleaf

## Thymeleaf 란?

웹 및 웹이 아닌 환경 모두에서 작동할 수 있는 Java XML/XHTML/HTML5 템플릿 엔진

## 기능

### Evaluating Expressions as Booleans

하단에 해당되지 않을 경우 true 로 판단한다.

- null
- the boolean value false
- the number 0
- the character \0
- the strings "false", "off" and "no"
