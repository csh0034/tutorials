# Spring Cloud Gateway
SCG 는 스프링 5, 스프링 부트 2, 프로젝트 리액터를 기반으로 API Gateway 를 구축할 수 있다.   
스프링 클라우드 게이트웨이의 주목적은 간단하면서도 확실한 방법으로 요청을 다른 API 로   
라우팅하고, 보안, 모니터링/메트릭, resiliency 같은 횡단 관심사를 모아 처리하는 것이다.
