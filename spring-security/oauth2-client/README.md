# Spring Security OAuth2 Client

## OIDC(OpenID Connect) 란?

- OAuth 2.0 프로토콜의 최상위 레이어와 동일한 레이어다.   
- OAuth 2.0을 확장하여 인증 방식을 표준화한다.
